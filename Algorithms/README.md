## Mathematical Functions
### Greatest Common Divisor（最大公约数）
* 最大公因数，也称最大公约数、最大公因子，指两个或多个整数共有约数中最大的一个；
* 算法：**辗转相除法**（**欧几里德算法**）；

### Euler Totient Function（欧拉函数）
* 在数论，对正整数n，欧拉函数是小于等于n的数中与n互质的数的数目；
